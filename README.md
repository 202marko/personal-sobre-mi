#  Proyecto: Mi Sitio Web Personal (Sobre Mí)

> **Estado:**  En Desarrollo

Este proyecto es la implementación técnica de la sección "Sobre Mí" de mi portafolio. Es un sitio web estático que sirve como mi tarjeta de presentación digital, integrando mi formación en la **Universidad CENFOTEC** y mi trayectoria profesional.

###  Detalles Técnicos
- **Desarrollo:** HTML5, CSS3 (siguiendo metodología BEM para estilos).
- **Enfoque:** Mobile-first y accesibilidad web.
- **Despliegue:** GitHub Pages.

###  Lo que aprendí/reforcé en este proyecto
- Estructuración de layouts con CSS Grid y Flexbox.
- Uso de variables CSS para temas de color.
- Optimización de imágenes para carga rápida.
- Gestión de versiones con Git y GitHub.

###  Contenido del Sitio
- Perfil profesional (Desarrollo + Ventas B2B).
- Stack tecnológico actual.
- Enlaces a redes profesionales y CV.
